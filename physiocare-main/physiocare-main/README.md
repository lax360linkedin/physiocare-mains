# PhysioCare

> Front-end for a physiotherapy clinic built with React, Vite and Tailwind CSS.

## Overview

PhysioCare is a responsive single-page front-end application for a physiotherapy clinic. It uses modern React (with Vite) and Tailwind CSS for fast development, plus Framer Motion for UI animations and React Router for navigation.

## Features

- Responsive, component-driven UI
- Built with React + Vite for fast dev/reload
- Styled with Tailwind CSS
- Smooth UI animations via Framer Motion
- Iconography with Lucide
- Routing with React Router

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- React Router

## Project Structure

Top-level folders you will work with:

- `src/` — application source
  - `components/` — UI components grouped by feature
  - `common/` — shared small components (buttons, titles, etc.)
  - `data/` — static data used by the UI
  - `pages/` — route pages (`Home`, `Contact`)

## Requirements

- Node.js (18+) and npm (or yarn/pnpm)

## Quick Start

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build locally:

```bash
npm run preview
```

Run linter:

```bash
npm run lint
```

## NPM Scripts

- `dev` — start Vite dev server
- `build` — produce production build
- `preview` — locally preview production build
- `lint` — run ESLint

## Notes for Developers

- Tailwind is included as a dependency; the project uses the `@tailwindcss/vite` plugin for integration with Vite.
- Components live under `src/components` and `src/common` — follow existing patterns when adding new UI pieces.
